# 👽 Interactive Zoog (Processing ver.)

> ### 3줄 요약
>
>   - **인터랙티브 아트**: Processing(Java)으로 제작된, 사용자의 입력에 실시간으로 반응하는 Zoog 캐릭터입니다.
>   - **다양한 상호작용**: 마우스와 키보드 입력에 따라 Zoog의 시선, 표정, 색상, 크기 등이 다채롭게 변화합니다.
>   - **쉬운 실행**: Processing IDE만 있으면 누구나 쉽게 실행하고 경험할 수 있습니다.

-----

## ✨ 주요 기능

  - **마우스 인터랙션**: 마우스 위치에 따라 몸통과 시선이 따라 움직이며, 클릭 또는 드래그 시 표정과 색상이 바뀝니다.
  - **키보드 단축키**: R(리셋), S(스크린샷), 1/2/3(테마 변경), +/-(크기 조절) 등 다양한 단축키를 지원합니다.
  - **실시간 효과**: 캐릭터의 움직임에 따라 파티클, 이동 흔적(트레일), 반짝임 등 시각적 효과가 적용됩니다.

-----

## 🖥️ 실행 방법

1.  [Processing](https://www.google.com/search?q=https://processing.org/download)을 설치한 후, `InteractiveZoog.pde` 파일을 엽니다.
2.  상단의 ▶ **Run** 버튼을 클릭하여 실행합니다.

-----

## ⌨️ 조작법 (예시)

| 입력 | 기능 |
| :--- | :--- |
| **마우스 이동** | Zoog의 시선과 몸통 방향 조절 |
| **마우스 클릭/드래그** | 표정 변경 또는 색상 랜덤화 |
| **R** | 모든 상태 초기화 (리셋) |
| **S** | 현재 화면 스크린샷 저장 |
| **1, 2, 3** | `classic`, `neon`, `mono` 테마로 전환 |
| **+/-** | 캐릭터 크기 조절 |

-----

## 🧠 개발 노트

  - **렌더링 순서**: `바디 → 얼굴 → 눈/입 → 이펙트` 순서로 렌더링하여 레이어 순서를 보장했습니다.
  - **입력 처리**: 사용자 입력을 받는 이벤트 로직과 캐릭터의 상태를 관리하는 로직을 분리하여 코드의 유지보수성을 높였습니다.
  - **애니메이션**: 보간(lerp) 함수를 사용하여 수치 변화를 부드럽게 만들어 자연스러운 움직임을 구현했습니다.
