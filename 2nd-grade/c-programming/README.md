## 🚀 실행

GCC 컴파일러가 설치된 환경(जैसे, Linux, macOS, Windows의 WSL/MinGW)에서 아래 명령어를 통해 각 코드를 컴파일하고 실행할 수 있습니다.

```bash
# 1. C 소스 파일을 컴파일하여 실행 파일 생성
# 예시: gcc case-converter.c -o case-converter
gcc [파일명].c -o [원하는 실행 파일명]

# 2. 생성된 실행 파일 실행
# 예시: ./case-converter
./[원하는 실행 파일명]
```
